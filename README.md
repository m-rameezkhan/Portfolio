# Muhammad Rameez Khan Portfolio

Fresh React + Vite rebuild of the portfolio with:

- single-page modern layout
- dark and light mode
- responsive sections for about, skills, journey, projects, services, FAQ, and contact
- animated project filters and scroll-based reveals
- static resume PDF download

## Run locally

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

## Notes

- The contact form opens a pre-filled email draft in the user's default mail app.
- Resume downloads are served from `public/resume/resume.pdf`.
- Project screenshots used in the build are stored in `public/images/projects`.
